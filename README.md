# HireIQ — AI Resume Screening System 🎯

![Version](https://img.shields.io/badge/Version-2.0-blue?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Capstone_Project-green?style=for-the-badge)

**HireIQ** is a premium, AI-driven resume screening and candidate ranking application designed to streamline the recruitment process. Using Natural Language Processing (NLP) and TF-IDF scoring, it analyzes resumes against job descriptions to surface the most qualified candidates instantly.

---

## ✨ Key Features

- **🤖 AI Resume Parsing:** Automatically extracts skills, experience, and education from PDF, DOCX, and TXT files.
- **🧮 TF-IDF Scoring Engine:** Ranks candidates using term frequency-inverse document frequency and cosine similarity to ensure precise matching.
- **⚖️ Custom Weighting:** Recruiters can adjust the importance of Skills, Experience, Education, and Keywords to fit specific hiring needs.
- **🛡️ Bias Detection:** Includes a fairness audit tool to monitor demographic parity and ensure equitable screening.
- **📈 Insights Dashboard:** Real-time analytics on candidate quality, skill gaps, and pipeline velocity.
- **🌙 Glassmorphism UI:** A stunning, modern dark-themed interface with smooth transitions and interactive widgets.

## 🚀 Getting Started

### Prerequisites
No complex installation required! This is a client-side application.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/lalasa-thetakali/HireIQ---AI-Resume-Screening-System.git
   ```
2. Navigate to the project folder:
   ```bash
   cd HireIQ-v3.3-Organized
   ```
3. Open `index.html` in your browser or run a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   ```

## ⚙️ How It Works

1. **Upload:** Drag and drop resumes into the upload zone.
2. **Configure:** Create a Job Description and set the desired skills.
3. **Analyze:** Run the AI Screening engine.
4. **Rank:** View the ranked list of candidates with detailed score breakdowns.
5. **Shortlist:** Select your top picks and export the results.
