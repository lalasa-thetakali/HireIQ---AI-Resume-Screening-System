🤖 What is HireIQ?
HireIQ is an AI-powered Resume Screening & Candidate Ranking System built as a Major Capstone Project for the Artificial Intelligence internship at LaunchED. It automates the tedious process of manually reviewing resumes — analysing each candidate against a job description and ranking them by relevance using real NLP techniques.
⚙️ How It Works
📄
1. Upload Resumes
Upload PDF, DOCX, or TXT resumes. The system parses each file and extracts skills, experience, education, and certifications.
💼
2. Set a Job Description
Create a job description with required skills and keywords. This becomes the target profile the AI matches candidates against.
🧮
3. AI Scoring (TF-IDF)
Each resume is scored using TF-IDF + Cosine Similarity across five weighted dimensions: Skill Match, Experience, Education, Keywords, and Certifications.
🏆
4. Ranked Results
Candidates are ranked by composite score. Choose to display Top 3, 5, 10, or All. Shortlist the best fits and export as CSV or PDF.

HTML5 CSS3 (Custom Properties) Vanilla JavaScript TF-IDF Algorithm Cosine Similarity NLP Preprocessing Responsive Design PDF / CSV Export

Upload Guidelines

PDF, DOCX, TXT and RTF formats are supported. For best results, use text-based PDFs (not scanned images). Maximum file size: 5MB per file.
✅ Best Practices ▼
Ensure resumes contain clear section headings (Education, Experience, Skills). Structured formats improve parsing accuracy significantly.
🔒 Privacy & Data ▼
All resume data is processed locally in your browser. No data is sent to external servers. You can clear all data at any time.

Bias Monitoring Active
